Submission Files

The final submission package includes the following core files:

  • predict.py - The inference script that loads the saved model and writes end-of-turn probabilities to predictions.csv.
  • features_ext.py - The rigorous feature extraction pipeline utilizing causality-bound resemblyzer prosody embeddings and handcrafted acoustic metrics.
  • train_model.py - The training pipeline containing our L1-regularized Logistic Regression feature selection and Random Forest classifier, alongside our 5-Fold GroupKFold CV protocol.
  • model.pkl - The final serialized model pipeline.
  • NOTES.md - A concise, technical markdown document explaining our feature engineering, model architecture, and validation strategy.
  • SUMMARY.html - This executive summary file detailing final performance metrics and included files.
  • RUNLOG.md - Historical iteration logs detailing the development and debugging process over the course of the project.
